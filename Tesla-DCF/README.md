Tesla Discounted Cash Flow (DCF) Valuation

Built a fully integrated 5-year DCF valuation model for Tesla using Excel and publicly available financial data.

Key components:
- Revenue and operating performance forecasts
- NOPAT and unlevered free cash flow modeling
- Working capital, depreciation, and capital expenditure assumptions
- Terminal value estimation using the Gordon Growth method
- WACC calculation using CAPM
- Enterprise value to equity value bridge

The model includes valuation visuals such as free cash flow trends, present value breakdowns, and an EV-to-equity waterfall.
