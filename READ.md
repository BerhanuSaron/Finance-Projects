Finance Projects Portfolio

This repository contains selected finance and analytics projects demonstrating financial modeling, valuation, portfolio analysis, and data visualization using Excel, Power BI, and Python.

Projects include:
- Discounted cash flow valuation and corporate finance modeling
- Portfolio risk and return analysis using real market data
- Budget vs actual performance analysis and financial dashboards
- Python-based financial analysis and visualization

Each project folder includes a short overview, methodology, and supporting files.
